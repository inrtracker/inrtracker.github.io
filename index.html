Local doctor viewer

Local in browser

Open source, local, no backend

This viewer is a static GitHub Pages file. The INR data is encrypted by the iOS app and decrypted locally in your browser only after the access code is entered. No patient data is uploaded to GitHub, stored in a database, or sent to a backend server.

The source code is public for transparency. Medical practices can inspect what the page does before using it.
